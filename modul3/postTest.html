Pengenalan HTML
HTML adalah bahasa markup yang umumnya digunakan untuk menyusun bagian paragraf, heading, maupun link pada halaman web. HTML menyediakan kerangka dasar untuk sebuah website, memungkinkan kita menampilkan teks, gambar, tautan, tabel, form, dan elemen-elemen multimedia lainnya dalam browser.
